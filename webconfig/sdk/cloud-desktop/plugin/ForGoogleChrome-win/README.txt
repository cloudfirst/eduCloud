中文版：

如何在win7+google chrome环境下进行配置

步骤一：双击运行 luhyavm-handler-register.reg 文件
步骤二：把 luhyavm-windows-handler.exe 文件拷贝到 c:\windows\system32\目录下

然后关闭google chrome浏览器，并重新打开即可









英文版：
How to install luhyavm://ip:port protocol handler on win7+google chrome

1. download and install registry file : luhyavm-handler-register.reg
2. download and copy client/web/luhyavm-handler/luhyavm-windows-handler.exe to c:\windows\system32\

then open google-chrome browser, now it can handler properly on link like luhyavm://ip:port.
